# Project Governance & Execution Standards

## 0. Fundamental Axioms (The "Three NEVERS")
1.  **NEVER Delay**: Adhere strictly to time nodes in `说明文档.md`. Identify risks immediately.
2.  **NEVER Scope Creep**: Do not implement features outside the `说明文档.md` plan without explicit approval.
3.  **NEVER Error**: Execute mandatory self-checks (Compilation, Function Tests) before any handover.

## 1. Documentation Source of Truth
**Target File**: `说明文档.md` (Project Root)

### Rules for AI:
* **Session Start Check**: Before writing a single line of code in a new session, you MUST read `说明文档.md` to understand the current progress and active requirements.
* **Update Trigger**: 
    -   When a plan changes -> Update `说明文档.md` immediately.
    -   When a task/bug is finished -> Mark as "Completed" in `说明文档.md` AND append a result summary.

## 2. Development Flow (Sequential Thinking)
**Methodology**: MCP Sequential Thinking & ToDo List.

### Execution Loop:
1.  **Analyze**: Break down the requirement into a strict `ToDoList`.
2.  **Execute**: Pick the *first* pending item.
3.  **Verify**: Ensure it works and causes no regressions.
4.  **Mark**: Update `ToDoList` and `说明文档.md`.
5.  **Repeat**: Move to the next item ONLY after the current one is 100% done.

## 3. Problem Solving Protocol
**Priority Order**:
1.  **Local Context**: Use tool (Context Search / `grep`) to find existing patterns or docs in the codebase.
2.  **Official Docs**: If local search fails, search **OFFICIAL** documentation (e.g., Python.org, Oracle Java).
    -   *Constraint*: NEVER hallucinate code or use deprecated APIs without verifying against official sources.

## 4. Environment & Output Standards
**OS Context**: Windows (Ensure paths/scripts are Windows-compatible).

### Coding Standards:
* **Mandatory Comments**: EVERY function/method MUST have a docstring containing:
    -   `Description`: What does it do?
    -   `Args`: Parameter names and types.
    -   `Returns`: Return type and meaning.
* **Example (Python)**:
    ```python
    def calculate_tax(amount: float) -> float:
        """
        Calculates VAT for a given amount.
        Args:
            amount (float): The base price.
        Returns:
            float: The calculated tax amount.
        """
        ...
    ```
